# Video Splitter

This is a desktop application made using Python and FFmpeg, which enables users to split their big video files into segments depending on the size they desire. 
The application automatically determines the segment length, processes the videos using FFmpeg, and gives a simple user interface to select the source files and output folders among other preferences.

## Features
Easy-to-use Tkinter GUI
Ability to split videos into desired file size
Use of FFmpeg for video processing
Progress monitoring when splitting
Supports many video file formats
Automatic determination of chunk length

## Technologies Used
Python
Tkinter
FFmpeg
